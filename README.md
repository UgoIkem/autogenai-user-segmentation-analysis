# AutogenAI User Segmentation & Engagement Analysis

## Project Overview

This project analyzes anonymized product usage data from an AI platform to understand how feature adoption relates to user engagement and retention.

The objective was to identify behavioral differences across user segments and provide actionable recommendations that could improve product adoption and long-term retention.

## Business Questions

* How engaged are users on the platform?
* How does retention vary across different user groups?
* Do users who adopt more features retain better?
* Which platform features drive the highest engagement?
* How do customer segments differ in usage behavior?

## KPIs Analysed

* Daily Active Users (DAU)
* Days Online per User (Per Month)
* Transformations per User (Per Month)
* Month-over-Month Retention Rate

## Segmentation Approach

Users were segmented based on the number of distinct platform features used:

* Single Feature Users
* Two Feature Users
* Multi-Feature Users

## Key Findings

* Multi-feature users retained at 65%, compared with 36% for single-feature users.
* Multi-feature users averaged 20.16 active days compared with 5.32 for single-feature users.
* Multi-feature users generated approximately three times more transformations than single-feature users.
* Research was the most-used feature across most customer segments.
* SMEs demonstrated slightly higher retention than Enterprise customers.

## Business Recommendations

1. Target single-feature users with feature discovery campaigns.
2. Encourage two-feature users to adopt a third feature through workflow recommendations.
3. Position Research as a key activation feature during onboarding.
4. Tailor onboarding experiences based on organization type.
